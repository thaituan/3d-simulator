
## スタイルガイド

統一されたUIを維持するためのルール。

### カラー

| 用途                 | 変数                    | 値        |
| -------------------- | ----------------------- | --------- |
| プライマリー         | `--color-primary`       | `#009E96` |
| プライマリー(hover)  | `--color-primary-hover` | `#008a83` |
| テキスト             | `--color-text`          | `#333333` |
| テキスト(補助)       | `--color-text-light`    | `#666666` |
| 背景                 | `--color-bg`            | `#ffffff` |
| 背景(セカンダリ)     | `--color-bg-secondary`  | `#f5f5f5` |
| ボーダー             | `--color-border`        | `#e0e0e0` |

- **背景色は白（#ffffff）を基本とする。グラデーション禁止**
- 色のハードコーディング禁止。必ず CSS 変数を使用する

### タイポグラフィ

| サイズ名 | 変数               | px   | 用途               |
| -------- | ------------------ | ---- | ------------------ |
| xs       | `--font-size-xs`   | 12px | キャプション       |
| sm       | `--font-size-sm`   | 14px | 補助テキスト       |
| md       | `--font-size-md`   | 16px | 本文               |
| lg       | `--font-size-lg`   | 20px | セクション見出し   |
| xl       | `--font-size-xl`   | 24px | ページ小見出し     |
| 2xl      | `--font-size-2xl`  | 32px | ページタイトル     |

- font-weight は `400`（通常）、`500`（中）、`600`（太）、`700`（見出し）のみ使用

### スペーシング

| サイズ名 | 変数              | px   |
| -------- | ----------------- | ---- |
| xs       | `--spacing-xs`    | 4px  |
| sm       | `--spacing-sm`    | 8px  |
| md       | `--spacing-md`    | 16px |
| lg       | `--spacing-lg`    | 24px |
| xl       | `--spacing-xl`    | 32px |
| 2xl      | `--spacing-2xl`   | 48px |

- マジックナンバー禁止。スペーシングには必ず CSS 変数を使用する

### レスポンシブ

| ブレークポイント | 幅       | 対象         |
| ---------------- | -------- | ------------ |
| mobile           | ≤ 768px  | スマートフォン |
| desktop          | > 768px  | PC           |

- **モバイルファースト or デスクトップファーストの混在禁止** — デスクトップファーストで `max-width` を使用する
- シミュレータページはモバイル時に縦積みレイアウト（Canvas → サイドパネル）に切り替え

### コンポーネントスタイル

1. **CSS Modules を使用** — グローバル CSS はリセットとCSS変数定義のみ
2. **クラス名は camelCase** — `styles.canvasWrapper` のように
3. **インラインスタイルは原則禁止** — 動的な値のみ許可（例: 3Dオブジェクトの位置座標）
4. **ネストは最大2階層** — セレクタの深いネストは可読性を損なう
5. **`!important` 禁止**

### ボタン・インタラクション

- ボタンホバー時は `background-color` の変化のみ
- `cursor: pointer` をすべてのクリック可能要素に設定
- フォーカス時のアウトラインは削除しない（アクセシビリティ）
- border-radius は `--border-radius`（8px）または `--border-radius-sm`（4px）を使用

### シャドウ

| 用途     | 変数           | 使用場面                   |
| -------- | -------------- | -------------------------- |
| 軽い影   | `--shadow-sm`  | カードホバー前             |
| 中程度   | `--shadow-md`  | カードホバー時、ドロップダウン |
| 大きい影 | `--shadow-lg`  | モーダル、管理者パネル     |
